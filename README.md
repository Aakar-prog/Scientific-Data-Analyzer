# Scient# Scientific Data Analyzer

A simple Python toolkit for basic scientific data analysis, including:

- CSV data loading
- Mean and standard deviation computation
- Linear regression
- Data visualization

This project is intended as a learning-oriented scientific analysis framework.

---

## Features

- Load experimental data from CSV files
- Compute statistical quantities (mean, standard deviation)
- Perform linear regression
- Plot data and fitted curves
- Unit tests for core functionality
